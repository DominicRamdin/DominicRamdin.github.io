This is my Scratch Project


This scratch game is a basic ping pong game against yourself. Use the mouse to move the paddle. Once the ball makes contact with the paddle it will bounce in the opposite direction. The game will get progressively harder as time passes up to level 3.

This is my MITAppInventor project



The MITAppInventor project I created is a picture shuffler. When you tap the tile on the screen the tile will flip over and reveal an image. This process will continue for all 6 tiles. once you do it 6 times it will reset. However, when it resets the order of the pictures is randomized.

This is my Python Interactive Fiction project

In my interactive fiction project you are a brave normal person who likes treasure and embarks on a quest to find said treasure. On your way you will meet a troll that guards the treasure. Instead of asking you riddles, he asks really confusing logic questions. If you answer all questions correct the troll will grant you access to the treasure.
